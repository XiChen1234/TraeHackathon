# 真实时薪计算器 - 打工人的「Reality Check」

## 📣 项目简介

这是一款专为**牛马打工人**设计的「真实时薪计算器」，拒绝老板画的饼，拒绝自我安慰式计算！把房租、通勤、摸鱼、KPI压力这些隐形成本都算进去，让你看清自己到底是在赚钱还是在「付费上班」。还能解锁各种整活成就，主打一个「笑中带泪」的真实体验。

在这里，你会发现自己时薪可能买不起一杯奶茶，加的班比吃的饭还多。摸鱼时间、通勤成本、KPI压力……全给你算得明明白白。用最扎心的数据告诉你：打工人的时薪，到底值几斤白菜？

算完别哭，我们只是帮你做reality check！

## ✨ 功能特点

### 🧮 真实计算
- **多种薪酬类型**：支持时薪、日薪、月薪、年薪
- **灵活工作制度**：双休、大小周、单休、上四休三、无休，自定义天数
- **精确时间计算**：考虑休息时间、摸鱼时间、通勤时间
- **生活成本核算**：房租、餐费等隐形成本
- **工作压力评估**：KPI等级、工作氛围、同事关系、工位环境

### 🎯 扎心对比
- 时薪可购买物品对比（奶茶、咖啡、白菜...）
- 直观展示你的「真实价值」
- 笑中带泪的结果呈现

### 🏆 成就系统
- 解锁各种整活成就
- 收集属于你的打工人勋章
- 分享你的「战绩」

### 📊 透明计算
- 详细展示计算过程
- 让你明白每一分钱的来去
- 拒绝黑盒计算，拒绝自我欺骗

## 🎮 核心功能说明

### 1. 薪酬信息
- 选择薪酬类型（时薪/日薪/月薪/年薪）
- 输入对应薪酬金额

### 2. 工作制度
- 选择常见工作制度或自定义天数
- 精确计算每月工作天数

### 3. 工作时间
- 设置每日工作开始和结束时间
- 调整休息时间、摸鱼时间、通勤时间

### 4. 生活成本
- 输入月均房租
- 设置日均餐费

### 5. 工作评估
- 选择KPI等级
- 评估工作氛围、同事关系、工位环境

### 6. 查看结果
- 获得真实时薪
- 查看可购买物品对比
- 解锁成就
- 查看详细计算过程

## 🏅 成就系统

我们设计了各种有趣的成就，让你的计算过程充满惊喜：

- **「温饱线挣扎」**：时薪低于XX元
- **「奶茶自由」**：时薪足够买一杯奶茶
- **「咖啡续命」**：时薪足够买一杯咖啡
- **「摸鱼大师」**：摸鱼时间超过工作时间XX%
- **「通勤战士」**：每日通勤时间超过XX小时
- **「KPI奴隶」**：KPI等级为S级

更多成就等你解锁！
